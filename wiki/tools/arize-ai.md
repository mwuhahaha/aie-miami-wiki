## Why It Matters

Arize AI sits close to the production side of AI engineering, where agent behavior has to be inspected, evaluated, and improved rather than merely demonstrated. That makes it a useful lens for Laurie's interface taxonomy: MCP exposes live runtime capabilities such as search, execution, memory, or tool access, while skills package reusable procedures and context that an agent can carry across environments.

The broader engineering lesson is that agent infrastructure is not only about adding more tools. It is about deciding which capabilities should be live services, which should be portable instructions, and which interfaces are stable enough for teams to trust in production workflows.
